# TALENTBook

Comprehensive talent management platform for the Vietnamese entertainment industry.

## Quick Start

### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL 14+

### Installation

1. Clone the repository
2. Setup backend
3. Setup frontend
4. Run development servers
